# Suporte

Use as [issues do repositório](https://github.com/andrelucassvt/brain-flows/issues) para relatar bugs, incompatibilidades ou solicitar melhorias.

Inclua a plataforma usada (Claude Code ou Codex), a versão do plugin, a skill invocada, o prompt mínimo para reproduzir o problema e o comportamento observado. Não publique credenciais, código privado ou dados pessoais.
